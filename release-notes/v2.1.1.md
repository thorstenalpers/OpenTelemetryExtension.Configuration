# v2.1.1

_Released: 2026-06-21_

## Changed
- Refreshed the README copy and NuGet package metadata, and added an example
  `appsettings.json` (`docs/appsettings.Example.json`) showing the available
  telemetry configuration keys.
- Reworked OTLP exporter configuration internally: each signal (traces, metrics,
  logs) is now wired up through a dedicated helper instead of a stringly-typed
  signal argument. No public API or runtime behavior change.
